---
title: A girl so pretty
description: A random thought about how people feel when things around them change...
date: 2026-07-15
tags: [Poetry]
---

Starting to live in a brand new city, wanted to save by being too petty… while I kept wishing for is a girl so pretty... 

⁠⁠Weekends going bored and waste, eating with the same ol’ taste… would never even sense the pity, only if I had a girl so pretty... 

⁠⁠Worrying about letting the clock run free, but wouldnt care if she asked for tea… i think how is she so witty, forgetting myself in this girl so pretty... 

⁠Acting like the things u do are bore, but honestly I couldnt ask for anymore… hurry and get me up in a jiffy, while all I feel is you’re a girl so pretty.
