Hey! This is my first blog post. It is a short guide to Markdown syntax, written while figuring out how to make a GitHub Pages site easier to maintain.

## How does Markdown work?

Markdown is plain text with lightweight syntax that a Markdown processor turns into HTML.

## Basic syntax

Use headings, lists, links, images, code, blockquotes, tables, and fenced code blocks to give notes a clear structure.
